嗨！我是張宇霆

擁有 Figma 專業設計能力與 Google UX 認證，擅長運用社會學研究方法進行需求洞察。具備前端開發背景，致力於將美學設計轉化為兼具技術可行性與互動體驗的方案。此外，具備 Python 爬蟲與 API 串接實作經驗，能理解資料流邏輯並整合至前端開發流程，打造流暢且直覺的數位產品體驗。

從社會學邏輯到前端實作

社會學的研究洞察是我的底蘊。我習慣於解析複雜的社會結構與非結構化數據，並將這種「洞察力」與「架構感」，轉化為開發時對代碼穩定性與使用者行為的精準控管。

Phase 01: Sociology

分析洞察：處理非結構化邏輯

擅長處理「非結構化資料」與進行「深度訪談」。這份訓練賦予了我極強的 UX 易用性分析直覺，能從混亂的資訊中提煉出行為邏輯，並預判使用者在介面上的潛在阻礙。

Phase 02: Digital Strategy

數據導向：掌握商業轉換邏輯

深耕廣告數據領域，將 SEO 與用戶留存思維內化。我認為前端開發不僅是實踐視覺，更是為了建立具備商業價值的轉換邏輯，確保每一行代碼都能優化產品的留存與轉化率。

Phase 03: Engineering

前端實踐：打造雜誌質感介面

以 Vue.js 與原生 CSS 為技術利刃，追求極致的排版與互動。我致力於將雜誌印刷的質感帶入數位產品，在確保效能與架構穩定的同時，提供最具渲染力的閱讀與操作體驗。
